# ScholarKit 🎓
### Multi-Vendor E-Commerce Database System

ScholarKit is a full-stack platform designed to streamline school uniform procurement. 

### Key Database Features:
* **ACID Transactions:** Managed via MySQL Stored Procedures for secure checkouts.
* **Normalization:** Database architecture strictly follows 3rd Normal Form (3NF).
* **Automated Logic:** Triggers for price auditing and low-stock notifications.
* **Security:** AES-256 encryption for sensitive user data.
* **Analytics:** Advanced Window Functions for best-seller insights.

### Tech Stack:
* **Frontend:** React.js, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MySQL
* **Payments:** Razorpay API Integration
